Attribute, Klassen und Ids

Attribute:

Attribute werden immer im Starttag definiert

Sind Schlüssel="Wert" Paare, die direkt in den Tag reingeschrieben werden.

Die meisten Attribute sind schon vorgegeben und können pro Element bei MDN nachgeschaut werden.

Ändern häufig das Verhalten oder die Darstellung

Unterscheiden zwischen booleschen Attributen und allen anderen

Boolesche Attribute gelten als true, sobald sie im Starttag stehen

Klassen:

Sind Sonderformen von Attributen

Werden wie folgt gesetzt < p class="Klassenname1 Klassenname2 usw..." >

Diese Klassen dienen zum styling mittels CSS und zum ansprechen der Elemente mittels JavaScript

Klassennamen sind case-sensitive

Mehere Elemete können der selben Klasse angehören

Die Klassennamen sollen der Funktion des Elementes entsprechen und nicht das Aussehen beschreiben

Id:

Sonderform der Attributes

Werden wie folgt gesetzt < p id="idName" >

Diese Klassen dienen zum styling mittels CSS und zum ansprechen der Elemente mittels JavaScript

Sind case-sensitive

Sollten einzigartig sein

Die Id sollte auch die Funktion des Elementes beschreiben
